# computor

It's a calculator

```
stack build
stack run -- --help
stack run -- --poly "3X + 2X^2 = 4"
stack run -- --poly "X^2 + 2X + 2 = 0"
stack run -- --repl --help
stack run -- --repl example/example1.txt
```

- readline (readline-devel)
